## 学习笔记
## 12、动态规划

- 将一个复杂的问题分解成简单的子问题，用一种递归的方式
- 分治+最优子结构
- 对比
  
  ​	- 动态规划、递归、分治没有根本区别（关键看有无最优子结构）

  ​	- 共性：找重复子问题

  ​	- 差异性：最优子结构、中途可以淘汰次优解

- 动态规划关键点

  ​	- 最优子结构 opt[n] = best_of(opt[n-1], opt[n-2], …)
  
  ​	- 储存中间状态：opt[i]
  
  ​	- DP方程
   
        一维：Fib: opt[i] = opt[n-1] + opt[n-2]
        二维：opt[i,j] = opt[i+1][j] + opt[i][j+1] (且判断a[i,j]是否空地）
